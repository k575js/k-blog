    # Ajax基本概念和作用

    1、Ajax定义 Ajax 即“Asynchronous Javascript And XML”（异步 JavaScript 和 XML）的缩写，意思就是用JavaScript执 行异步网络请求。 通俗解释：在网页上利用JavaScript编写代码操作XMLHttpRequest对象来与服务器进行数据交互的方式就是Ajax 2、典型应用场景： Ajax的局部更新：

    # jQuery中 $.get()方法 请求服务器Api接口获取 数据

    $.get(
      'http://127.0.0.1:3001/getHeroSkin', // 请求的服务器Api接口
      null,                                // 参数对象
      function(resData){                   // 回调函数
         console.log(resData);             // 处理服务器响应回来的数据
      }
);

    # URL地址的组成

    URL的组成：协议，服务器名称，路径，参数 1、什么是URL 统一资源定位系统（uniform resource locator;URL）是用来标记网络上一个资源唯一存放位置，浏览器通过URL就可以成功访问到该资源

    2、常见的URL举例

    http://www.itcast.cn/subject/about/index.html
http://157.122.54.189:9092/nc/course/courseDetial/getSectionQAByPage/3?pageIndex=3&pageSize=1

    1. Ajax概念和作用 异步对象XMLHttpRequest，局部更新

    3. URL地址组成 通讯协议、服务器名称、路径、参数

    # Api接口和接口文档

    1、什么是Api接口 API（Application Programming Interface，应用程序接口）是预先定义好的一组接口，每一个API都有自己特定的功能，供需要它的程序调用。

    2、Ajax课程中的Api接口 前端工程师将来打交道最多的是Http API接口，此种接口会提供一个URL地址暴露在互联网上，可以通过编写Ajax代码去与之交互来完成功能

    # Api接口文档组成

    1、接口基本信息说明 接口地址：接口暴露出来的URL地址 请求方式：GET/POST 请求示例：接口的完整请求URL举例

    2、接口请求参数说明 以表格的形式描述请求接口时需要传入的参数 方便调用者清晰了解每一个参数的作用

    3、接口响应参数说明 以表格形式描述服务器响应回来每个数据代表的含义
